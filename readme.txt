== Description ==

Whatsapp Checkout Order for WCMF Multi Vendor is a custom Wordpress plugin that streamline the checkout process for your customer through Whatsapp. This plugin allows your customer to checkout and send the order summary to vendor's whatsapp number that has been registered beforehand "Whatsapp Number" WCFM custom field.


How does it work? 

The plugin adds a button on the Woocommerce checkout page so your customers can submit their orders through each vendor's WhatsApp. The vendor's will receive an order summary including product name & amount, shipping method, total payment, etc.


== HOW TO SHOW WA BUTTON ON CHECKOUT (READ CAREFULLY) ==
As Admin/Store Manager
1. Fill out your vendor's Whatsapp number in Vendors -> Store Setting -> Whatsapp Number. Use your international number format without the Plus sign such as 62812345678
2. Right below it, Set Show Whatsapp Number on Checkout to YES

As Vendor
1. Fill out your vendor's Whatsapp number in Store Setting -> Store -> General Setting -> Whatsapp Number. Use your international number format without the Plus sign such as 62812345678
2. Right below it, Set Show Whatsapp button on Checkout to YES

Thats it. Enjoy! If you like this plugin, consider donating to help us continue improving this plugin through paypal.me/aditbaco 

Thank you!

== Installation ==

= Installing manually: =

1. Unzip all files to the `wp-content/plugins/wcfm-whatsapp-checkout` directory
2. Log into WordPress admin and activate the 'WCFM Whatsapp Checkout' plugin through the 'Plugins' menu
3. Enjoy your new checkout!

= Updating new version =
1. Deactivate your old WFCM Whatsapp Checkout in Plugins setting and delete it.
2. Upload and activate the new version. Don't worry your Vendors Whatsapp number won't be lose.

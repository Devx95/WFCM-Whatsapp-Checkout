== Description ==

Whatsapp Checkout Order for WCMF Multi Vendor is a custom Wordpress plugin that streamline the checkout process for your customer through Whatsapp. This plugin allows your customer to checkout and send the order summary to vendor's whatsapp number that has been registered beforehand "Whatsapp Number" WCFM custom field.


How does it work? 

The plugin adds a button on the Woocommerce checkout page so your customers can submit their orders through each vendor's WhatsApp. They will receive a personalized message with the product name, shipping method, total payment, etc.


== HOW TO USE (READ CAREFULLY) ==

1. Navigate to WCFM Store-Manager -> Settings -> Vendor Registration and then create a custom field with below details :

    Field Type: Number
    Label : Whatsapp Number
    Help Content : 62812345678 (adjust the phone example to your international number format without the Plus sign)

Please note that "Whatsapp Number" is case sensitive or IT WON'T WORK!

2. Fill out your vendor's Whatsapp number in Vendors -> Profile. Use your international number format without the Plus sign such as 62812345678


Thats it. Enjoy!


Pls note that this is my custom plugin to suit my needs. You may also adjust it to suit your needs.

3. Go to *Loco Translate > Home* in the left-hand menu to start translating
